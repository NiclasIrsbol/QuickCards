# QuickCards
Flashcard app - Allows a user to create, edit, manage and store mulitple sets of flashcards. The app was made using Java, and JavaFX for GUI, and the sets of flashcards are stored locally as .JSON files.

# How to run?
To run the application, you should run the class *mainScene.java*.

# Motivate behind this project.
I made this simply because I couldn't find any good alternative online, and the ones I did find either wanted me to spend money, or I was only able to make 5-10 cards. So, I just made my own :).
